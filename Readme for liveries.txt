For the Add-on:
- Add-on should be correctly installed
- Use OpenIV to add the Liveries to BMWe90.ytd
Location: mods\update\x64\dlcpacks\BMWe90\dlc.rpf\x64\vehicles.rpf\

For the Replace:
- To activate the livery support, drop "vehicles.meta" to: \mods\update\update.rpf\common\data\levels\gta5
- or edit it manually:
1. Edit "vehicles.meta" with OpenIV or Notepad:
2. Search "asterope", scroll down to Flags line and add: FLAG_HAS_LIVERY
3. Use OpenIV to add the Liveries to asterope.ytd

Important:
Liveries always need  to have correct numeration in the YTD file!
For example: e90_sign_1, _2, _3, _4 and so on.
Notice: Dont rename liveries inside the YTD file or they will not work!
